# three-types-modded

[![npm version](https://img.shields.io/npm/v/three-types-modded)](https://www.npmjs.com/package/three-types-modded)

TypeScript type definitions for [three-modded](https://www.npmjs.com/package/three-modded) ([GitHub](https://github.com/repalash/three.js-modded)), a modified fork of [three.js](https://github.com/mrdoob/three.js) used by [threepipe](https://github.com/repalash/threepipe) and [webgi](https://webgi.dev).

```bash
npm install @types/three@npm:three-types-modded
```
